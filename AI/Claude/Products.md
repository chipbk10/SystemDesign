# Claude Products Summary

## End User
Tools for individuals who want to chat with Claude.
Both use the same **claude.ai account and plan**.

### [claude.ai](https://claude.ai)
A website you open in your browser. You chat with Claude, that's it. Works on any device.
Has free and paid (Pro, Team, Enterprise, Max) tiers.

### [Claude Desktop App](https://claude.ai/download)
A Mac/Windows app you install on your computer. Same chat experience as claude.ai, but with extra capabilities like Computer Use (controlling your screen, browsing, etc.).
Requires a paid plan for extra capabilities.

---

## Developer
Tools for developers who want to build with or code alongside Claude.
Both use **platform.claude.com API credits** — unless you have a claude.ai Max plan, in which case you can log in directly with your claude.ai credentials instead.

### [platform.claude.com](https://platform.claude.com)
A website for developers. Where you manage API access, create API keys, monitor usage, set up projects, run experiments in the Workbench (a prompt testing playground), and handle billing for API usage.

### [Claude Code CLI](https://docs.anthropic.com/en/docs/claude-code/overview)
A terminal tool you install on your computer via npm. You interact with Claude by typing commands in the terminal. It can read your entire codebase, edit files, and run commands directly.

### [Claude Code VS Code Extension](https://marketplace.visualstudio.com/items?itemName=anthropic.claude-code)
The same thing as Claude Code CLI, but with a visual panel inside VS Code. No need to type terminal commands — everything is done through a GUI. The CLI is bundled inside the extension, so no separate installation needed.
