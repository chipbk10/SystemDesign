Your team has analyzed large volumes of user data and determined that customers primarily use Feature A. Your manager wants to spend the next quarter adding complex new features to Feature B, based on an informal request from a high-level executive. The data shows Feature B has only 5% usage.

**Question:** Rank the following actions from most effective to least effective:

* **A.** Agree to the manager's plan to avoid conflict and build the features for Feature B as quickly as possible.
* **B.** Quietly use your team's reserve capacity to enhance Feature A with high-impact fixes while still officially working on Feature B.
* **C.** Present the manager with the usage data (Feature A: 80%, Feature B: 5%) and propose a compromise to dedicate a small amount of time to Feature B while focusing most effort on Feature A.
* **D.** Immediately challenge the executive's request via email, copying your manager and other stakeholders to highlight the data.
