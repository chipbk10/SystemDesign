## ✅ TCO (Total Cost of Ownership)

- What it is: The overall cost of running workloads on AWS compared to on-premises.
- Includes: Hardware, software, maintenance, power, cooling, staff, and AWS service costs.
- AWS TCO Calculator: Helps estimate savings when migrating to AWS.


## ✅ Cost Optimization

- What it is: Strategies to reduce AWS costs without sacrificing performance.
- Examples:
  - Use **Right-Sizing** (choose correct instance size).
  - Use **Reserved Instances** or **Savings Plans** for predictable workloads.
  - Use **Auto Scaling** to avoid paying for unused capacity.
  - Use **Spot Instances** for flexible workloads.
  - Use S3 lifecycle policies to move data to cheaper storage tiers.
