**AWS Compute Optimizer** is a service that helps you **optimize your AWS resources for cost and performance** by using **machine learning-based recommendations**.

***

### ✅ **What it does**

*   Analyzes your usage patterns for:
    *   **EC2 instances**
    *   **Auto Scaling groups**
    *   **EBS volumes**
    *   **Lambda functions**
*   Suggests **right-sizing** (e.g., smaller instance type if underutilized).
*   Helps reduce **costs** and improve **efficiency**.

***

### **Key Benefits**

*   **Cost savings**: Avoid over-provisioning.
*   **Performance improvement**: Prevent under-provisioning.
*   **Easy to use**: Recommendations shown in AWS Console.

***

✅ **Example:**  
If you run an EC2 `m5.large` but only use 10% CPU, Compute Optimizer might recommend `t3.medium` to save money.
