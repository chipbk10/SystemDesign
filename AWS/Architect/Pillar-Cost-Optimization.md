### ✅ Cost Optimization Pillar (AWS Well-Architected Framework)

The **Cost Optimization pillar** focuses on **avoiding unnecessary costs and maximizing value**. It emphasizes:

*   **Right-sizing resources** – Use the correct instance types and sizes.
*   **Using pricing models effectively** – On-Demand, Reserved Instances, Spot Instances.
*   **Monitoring and controlling costs** – Track usage and set budgets.
*   **Leveraging managed services** – Reduce operational overhead.

Goal: **Spend only what you need while maintaining performance and reliability.**

***

### ✅ AWS Services for Cost Optimization

1.  **AWS Cost Explorer**
    *   Analyze and visualize cost and usage.
2.  **AWS Budgets**
    *   Set custom cost and usage budgets with alerts.
3.  **AWS Trusted Advisor**
    *   Provides cost optimization recommendations.
4.  **Amazon EC2 Spot Instances**
    *   Use spare capacity at lower cost.
5.  **Savings Plans & Reserved Instances**
    *   Commit to usage for discounts.
6.  **AWS Compute Optimizer**
    *   Suggests optimal instance types for workloads.
7.  **S3 Intelligent-Tiering**
    *   Automatically moves data to cost-effective storage tiers.
