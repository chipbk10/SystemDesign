**Amazon GuardDuty** is an **AWS threat detection service** that continuously monitors your AWS accounts, workloads, and data for **malicious activity and unauthorized behavior**.

### ✅ **Key Features**

*   **Threat Detection:** Uses machine learning, anomaly detection, and threat intelligence to identify potential security issues.
*   **Data Sources:** Analyzes AWS CloudTrail logs, VPC Flow Logs, and DNS logs.
*   **Findings:** Generates actionable security findings (e.g., compromised IAM credentials, unusual API calls, crypto-mining activity).
*   **Integration:** Works with AWS Security Hub, EventBridge, and can trigger automated responses via Lambda.

### ✅ **Why Use It?**

*   No infrastructure to manage (fully managed and serverless).
*   Helps meet compliance and security best practices.
*   Continuous monitoring without performance impact on workloads.
