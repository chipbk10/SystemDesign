## ✅ AWS Shield
- **What it is**: A managed DDoS protection service.
- **Purpose**: Protects AWS applications from Distributed Denial of Service attacks.
- **Types**:
  - Shield **Standard**: Free, automatic protection for all AWS customers.
  - Shield **Advanced**: Paid, adds extra protection, real-time attack mitigation, and 24/7 AWS DDoS Response Team support.

## ✅ AWS WAF (Web Application Firewall)

- **What it is**: A firewall for **HTTP/HTTPS traffic**.
- **Purpose**: Protects web apps from common exploits like SQL injection, XSS, and bot attacks.
- **How**: You define rules to allow, block, or monitor requests based on IP, headers, or patterns.
