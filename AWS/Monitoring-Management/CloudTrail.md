## ‚úÖ What is CloudTrail?
- AWS CloudTrail is a service that records API calls and actions made in your AWS account. It tracks who did what, when, and from where.
- Example: If someone launches an EC2 instance or modifies an S3 bucket policy, CloudTrail logs that event for auditing and security purposes.
- Captures **who**, **what**, **when**, **where**
