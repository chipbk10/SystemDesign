Amazon Lex is an **AI service for building conversational interfaces** (chatbots and voice bots) using **natural language understanding (NLU)** and **automatic speech recognition (ASR)**.  
Key features:

*   Understands **text and speech input**.
*   Integrates easily with **Amazon Connect** for contact centers.
*   Common use cases: **customer support bots, voice assistants, FAQ bots**.

Goal: **Create intelligent chatbots without building ML models from scratch.**
