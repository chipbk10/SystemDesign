Amazon Rekognition is an **AI-powered image and video analysis service**. It can:

*   Detect **objects, people, text, and activities** in images and videos.
*   Perform **facial analysis and recognition**.
*   Identify **inappropriate content** for moderation.
*   Extract **text from images** (OCR).

Goal: **Add computer vision capabilities to applications without building ML models from scratch.**
